# Security Policy

## Supported versions

This project currently supports the latest release on the `main` branch.

## Reporting a vulnerability

Please avoid filing public issues for security vulnerabilities.

Preferred channel:

- Use GitHub Security Advisories ("Report a vulnerability") for private disclosure.

What to include:

- Affected area (endpoint/module) and impact
- Steps to reproduce or proof of concept
- Any suggested mitigation or patch

We will acknowledge receipt when possible and work on a fix.
